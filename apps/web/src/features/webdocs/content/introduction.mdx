## Welcome to Ludocode

Ludocode is an open-source, interactive coding education platform designed to make learning to program accessible and engaging. Learners can explore, experiment, and with the admin app, build their own courses.

This documentation covers project setup, feature configuration, and the key systems that make up the application.

<Tip variant="note" title="Work in Progress">
  Documentation currently focuses on setup, features, and key areas. More
  detailed architecture documentation will follow.
</Tip>

## Quick Setup

### Prerequisites

- [Docker](https://docs.docker.com/get-docker/)
- [Node.js 22+](https://nodejs.org/)

<Tabs
  tabs={[
    {
      label: "Setup",
      content: (
        <div className="text-sm text-ludo-white space-y-4">

          <p className="font-semibold text-ludo-white-bright">1. Clone the backend and start the API &amp; database with Docker.</p>

          <CodeBlock language="bash">{`git clone git@github.com:jokerhutt/ludocode-backend.git

cd ludocode-backend
cp .env.example .env

docker compose -f docker-compose.db.yml up -d
docker compose -f docker-compose.ludocode.yml up -d --build ludocode-backend`}</CodeBlock>

          <p className="font-semibold text-ludo-white-bright">2. Clone the frontend, install dependencies, and start both apps.</p>

          <CodeBlock language="bash">{`git clone git@github.com:jokerhutt/ludocode.git

cd ludocode
npm install

# Web app

cd apps/web && cp .env.example .env && npm run dev &

# Admin app

cd ../admin && cp .env.example .env && npm run dev`}</CodeBlock>

          <p>
            The API will be available at <code>http://localhost:8080</code> with a pre-seeded demo user and course.
          </p>

          <p>

          </p>

          <p>
            The admin app will be available at <code>http://localhost:5174</code>.
          </p>

        </div>
      ),
    },
