## Overview

Ludocode lets you create and manage your own courses through the admin app. The backend handles diffing and versioning automatically, and ensures that user progress is preserved for untouched lessons.

## How a Course Works

In the admin editor, a course is a structured container that groups modules and lessons.

The hierarchy is:

- **Course**
- **Module**
- **Lesson**
- **Exercise**

A course defines metadata such as title, description, visibility, and language.  
It contains ordered modules. Modules contain ordered lessons. Lessons contain ordered exercises.

All ordering is explicit and controlled in the editor.

## Editing Courses

Course editing is snapshot based. The backend sends the current snapshot, and the frontend sends backa a new one.

When you publish a course, it can no longer be deleted. However, you can archive it. If archived, users already enrolled can still complete lessons in the course, but it will be invisible to new users.

Editing is made in such a way that if you edit a lesson, the users progress for that lesson is preserved.
