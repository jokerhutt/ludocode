## Welcome to Ludocode

Ludocode is an open-source, interactive coding education platform designed to make learning to program accessible and engaging. Learners can explore, experiment, and — with the admin app — build their own courses.

This documentation covers project setup, feature configuration, and the key systems that make up the application.

<Tip variant="note" title="Work in Progress">
  Documentation currently focuses on setup, features, and key areas. More
  detailed architecture documentation will follow.
</Tip>

## Tech Stack

| Layer        | Technology      | Purpose                                  |
| ------------ | --------------- | ---------------------------------------- |
| Framework    | React 19        | UI rendering                             |
| Routing      | TanStack Router | File-based, type-safe routing            |
| Server State | TanStack Query  | Data fetching, caching, mutations        |
| Forms        | TanStack Form   | Type-safe form state (admin app)         |
| Styling      | Tailwind CSS v4 | Utility-first CSS with design tokens     |
| Editor       | Monaco Editor   | Code editing in projects & admin         |
| Validation   | Zod             | Schema validation for all data contracts |
| Build        | Vite 7          | Dev server & production bundling         |
| Auth         | Firebase Auth   | Google, GitHub, Email/Password login     |
| Payments     | Stripe          | Subscription checkout & management       |
| AI           | Vercel AI SDK   | Chatbot streaming & AI components        |
| Animation    | Motion + Lottie | UI transitions & celebration effects     |
| DnD          | dnd-kit         | Drag-and-drop in admin builder           |
| Testing      | Playwright      | End-to-end browser testing               |
