## Welcome to Ludocode

Ludocode is an interactive coding education platform that makes learning to program fun and engaging. This documentation covers the internal architecture of the Ludocode frontend — its routing system, data-fetching patterns, feature flows, and project structure.

<Tip variant="tip" title="New to Ludocode?">
  Start with the **Directory Structure** article to understand how the codebase
  is organized, then explore the feature-specific pages.
</Tip>

### Tech Stack

| Layer        | Technology      |
| ------------ | --------------- |
| Framework    | React 19        |
| Routing      | TanStack Router |
| Server State | TanStack Query  |
| Forms        | TanStack Form   |
| Styling      | Tailwind CSS v4 |
| Editor       | Monaco Editor   |
| Validation   | Zod             |
| Build        | Vite            |
